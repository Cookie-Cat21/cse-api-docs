POST /companyInfoSummery

prices probe ok

Per-symbol quote (primary single-name source).

Used by UI: Company profile / quote widgets

Last probe status: 200 · expect [200]

curl

curl -sS -X POST 'https://www.cse.lk/api/companyInfoSummery' \
  -H 'Content-Type: application/x-www-form-urlencoded' \
  -H 'Origin: https://www.cse.lk' -H 'Referer: https://www.cse.lk/' \
  -d 'symbol={{symbol}}'

Known failures

  • GET → 405
  • POST JSON {symbol} → 400

Fields

  • reqSymbolInfo.symbol
  • reqSymbolInfo.name
  • reqSymbolInfo.id — Numeric stockId for companyChartDataByStock (not securityId)
  • reqSymbolInfo.lastTradedPrice
  • reqSymbolInfo.change
  • reqSymbolInfo.changePercentage
  • reqSymbolInfo.marketCap
